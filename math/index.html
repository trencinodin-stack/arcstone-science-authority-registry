Mathematical Primitives

Canonical primitive mathematical objects.

  • Constraint
    A restriction defining allowed states or transitions.
  • Admissibility
    Predicate determining whether an object or action is allowed.
  • Transition
    Mapping from one state to another.
  • State
    A complete description of a system at an instant.
  • Proof Object
    Typed artifact certifying a claim.
  • Judgment Lattice
    Partially ordered set of decision outcomes.
  • Certificate
    Signed proof object granting permission.
