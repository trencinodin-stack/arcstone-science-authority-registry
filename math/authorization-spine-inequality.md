# Authorization Spine Inequality

## Form
AUTHORIZED ⇔ min(g₁, g₂, …, gₙ) ≥ 0

## Meaning
An action is authorized only if all required admissibility margins are non-negative.

## Interpretation
Authorization is conjunctive.
The weakest admissibility dimension dominates.

## Consequences
- Single failure blocks action.
- Partial satisfaction is insufficient.
- Authorization is fail-closed.

## Non-Scope
- Optimization of margins
- Weighting of dimensions
