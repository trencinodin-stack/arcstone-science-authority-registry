Typed Existence-Admission Kernel

Kernel

Determines whether an entity, state, or action is legally admissible to exist within a governed system.

Function

This kernel evaluates admissibility predicates over typed states and transitions. If admissibility fails, existence is denied regardless of utility, intent, or optimization.

Governs

  • Existence permission
  • State admission
  • Action eligibility
